# 4. Просмотр результатов

После завершения интервью система формирует итоговый отчет.

Отчет доступен сразу после окончания сессии, а также сохраняется в истории прохождений.

---

## 4.1 Итоговый балл

Каждая сессия оценивается по шкале от 0 до 100.

Итоговый балл формируется на основе:

- соответствия ответов критериям сценария,
- наличия ключевых формулировок,
- отсутствия нежелательных выражений,
- общей структуры ответа.

---

## 4.2 Обратная связь

В отчете отображается:

- общая оценка прохождения,
- сильные стороны ответов,
- выявленные ошибки,
- рекомендации по улучшению.

Обратная связь формируется автоматически на основе критериев, заданных экспертом для выбранного сценария.

---

## 4.3 Просмотр истории интервью

Пользователь может открыть раздел «История».

В этом разделе отображаются:

- список всех пройденных интервью,
- дата прохождения,
- итоговый балл,
- статус сессии.

При выборе конкретной сессии открывается детальный отчет.
