# 2. Регистрация и вход

## 2.1 Регистрация

Для создания учетной записи:

1. Откройте страницу регистрации.
2. Введите имя и фамилию.
3. Укажите адрес электронной почты.
4. Придумайте пароль.
5. Подтвердите регистрацию.

После успешной регистрации пользователь может выполнить вход в систему.

---

## 2.2 Вход в систему

Для входа:

1. Перейдите на страницу авторизации.
2. Введите адрес электронной почты.
3. Введите пароль.
4. Нажмите кнопку «Войти».

При корректном вводе данных система перенаправит пользователя на главную страницу.
