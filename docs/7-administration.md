# 7. Администрирование системы

Раздел предназначен для пользователей с правами администратора.

Администратор отвечает за глобальное управление платформой.

---

## 7.1 Управление категориями

Категории используются для группировки сценариев по направлениям (например: IT, Маркетинг, Финансы).

Администратор может:

- создать новую категорию,
- изменить название существующей,
- удалить категорию.

Для создания категории:

1. Перейдите в раздел «Категории».
2. Нажмите «Добавить категорию».
3. Укажите название.
4. Сохраните изменения.

Удаление категории возможно только при отсутствии связанных сценариев.

---

## 7.2 Управление экспертами

Администратор может управлять учетными записями экспертов.

Доступные действия:

- назначение роли эксперта,
- удаление эксперта,
- ограничение доступа.

Изменение роли пользователя влияет на доступный ему функционал.

---

## 7.3 Настройка системы достижений

Администратор создает достижения, которые могут получать пользователи.

Для создания достижения:

1. Перейдите в раздел «Достижения».
2. Нажмите «Создать».
3. Укажите название.
4. Добавьте описание условия получения.
5. Загрузите иконку.
6. Сохраните.

Созданные достижения становятся доступными для автоматического начисления системой.

---

## 7.4 Модерация контента

Администратор может просматривать сценарии, созданные экспертами.

При необходимости администратор может:

- редактировать сценарий,
- отклонить публикацию,
- удалить некорректный контент.

Это обеспечивает качество материалов на платформе.
