# 6. Работа со сценариями

Раздел предназначен для пользователей с правами эксперта.

Эксперт может создавать и настраивать сценарии интервью, которые будут доступны для прохождения.

---

## 6.1 Создание сценария

Для создания нового сценария:

1. Перейдите в раздел «Сценарии».
2. Нажмите кнопку «Создать сценарий».
3. Укажите название сценария.
4. Выберите категорию.
5. Заполните описание и контекст интервью.
6. Сохраните изменения.

После сохранения сценарий можно дополнительно настроить.

---

## 6.2 Настройка контекста

Контекст определяет условия интервью:

- описание компании,
- должность,
- уровень кандидата,
- дополнительные требования.

Контекст используется системой при формировании вопросов.

---

## 6.3 Добавление критериев оценки

Критерии определяют, какие ответы считаются:

- желательными,
- нежелательными,
- обязательными для упоминания.

Для добавления критерия:

1. Откройте сценарий.
2. Перейдите в раздел «Критерии».
3. Выберите тип критерия.
4. Введите текст условия.
5. Сохраните изменения.

Критерии используются системой при анализе ответов.

---

## 6.4 Добавление примеров вопросов

Эксперт может добавить примеры вопросов, на которые система должна ориентироваться.

Для этого:

1. Перейдите в раздел «Примеры вопросов».
2. Нажмите «Добавить».
3. Введите текст вопроса.
4. Сохраните.

Примеры используются как ориентир при генерации новых вопросов.

---

## 6.5 Редактирование сценария

Эксперт может изменить:

- название,
- контекст,
- критерии,
- список примерных вопросов.

Изменения применяются ко всем новым сессиям.

---

## 6.6 Тестирование сценария

Перед публикацией сценарий можно протестировать.

В режиме тестирования:

- создается изолированная сессия,
- результаты не сохраняются в основной истории,
- можно проверять различные формулировки ответов.

Этот режим позволяет убедиться, что логика оценки работает корректно.
