# 3. Прохождение интервью

## 3.1 Выбор сценария

Перед началом интервью необходимо выбрать сценарий.

Сценарии сгруппированы по направлениям. Пользователь может просматривать список доступных сценариев и выбрать подходящий.

Для запуска интервью нажмите кнопку «Начать».

---

## 3.2 Интерфейс интервью

Интервью проходит в формате текстового чата.

В интерфейсе отображаются:

- сообщения системы (вопросы),
- поле для ввода ответа,
- кнопка отправки,
- индикатор статуса сессии.

---

## 3.3 Отправка ответа

1. Введите текст ответа в поле ввода.
2. Нажмите кнопку «Отправить».
3. Дождитесь следующего вопроса.

Ответ анализируется системой автоматически.

---

## 3.4 Завершение интервью

Интервью может быть завершено:

- автоматически после достижения финального этапа,
- вручную по инициативе пользователя.

После завершения формируется итоговый отчет.
